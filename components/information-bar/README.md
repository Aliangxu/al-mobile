# n22-information-bar
全局咨询导航

### API

| 参数 | 说明 | 类型 | 默认值 | 版本 |
|------|------|------|------|------|
| leftImg | 图标 | `String` | - | - |
| direction | 咨询滚动方向 | `Boolean` | 默认vertical-垂直滚动，horizontal-水平滚动 | - |
| informationList | 咨询数据 | `Array` | 数据结构参考组件内的default | - |
| url | 跳转链接 | `String` | - | - |
